# 상품 등록 구조
1. 뷰 - 아이템 컨트롤러 - 아이템서비스- 아이템폼디티오를 이용해서 아이템 객체를 만든다.
2. 만든 아이템 객체를 아이템 리포지토리에 저장을 하게되면 DB로 저장한다.(상품이름,수량 등이 먼저 들어간다. ) 
DB로 저장이 끝나게 되면 아이템 리포지토리 - 아이템 서비스로 되돌아간다. 
3. 아이템 서비스에서 아이템 이미지객체를 생성한다.
4.아이템 이미지 객체를 이용하여 아이템이미지 서비스로 간다. 
5. 바로저장이 아니라 파일 서비스를 이용해서 파일 쓰기를 행한다. 서버 이미지 파일저장(C:/drama/item에 저장)아이템이미지 서비스로 들어가서 아이템이미지 리포지토리를 이용해서 DB에 저장하게 된다. (이미지 파일 개수만큼 반복한다)
6. 아이템 컨트롤러로 가서 뷰를 보여준다.
